About Isaac Compendium

What is Isaac Compendium?

Isaac Compendium is a comprehensive, community-driven guide designed to help both new and experienced players navigate the depths of this challenging roguelike adventure. Our mission is to provide accurate, detailed, and accessible information about items, characters, bosses, synergies, and strategies.

Features

  • Complete Items Database: Browse over 700+ items with detailed effects, unlock conditions, and item pool information.
  • Character Guides: Master all playable characters with comprehensive stat breakdowns and optimal strategies.
  • Boss Strategies: Learn attack patterns, weaknesses, and winning tactics for every boss encounter.
  • Synergy Discovery: Uncover powerful item combinations that can transform your runs.
  • Offline Support: Progressive Web App functionality allows offline access to all content.
  • Dark Mode: Eye-friendly dark theme with automatic detection based on your preferences.
  • Fast and Lightweight: No frameworks, no dependencies - just pure vanilla JavaScript for blazing-fast performance.

Technology

Isaac Compendium is built with modern web standards using vanilla HTML, CSS, and JavaScript. No frameworks, no build tools - just clean, performant code that works everywhere.

  • Semantic HTML5 for accessibility
  • CSS Grid and Flexbox for responsive layouts
  • ES6 Modules for organized JavaScript
  • Service Worker for offline functionality
  • Progressive Web App (PWA) capabilities

Accessibility

We are committed to making Isaac Compendium accessible to everyone. Our site follows WCAG 2.1 AA guidelines and includes:

  • Semantic HTML structure
  • ARIA labels and landmarks
  • Keyboard navigation support
  • High contrast color schemes
  • Screen reader compatibility
  • Skip links for easier navigation
